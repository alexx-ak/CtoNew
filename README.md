# VoxBox

VoxBox is a full-stack application consisting of an Angular frontend and a .NET backend.

## Project Structure

```
VoxBox/
├── frontend/          # Angular application (TypeScript)
│   ├── src/          # Source files for Angular app
│   └── public/       # Public assets
├── backend/           # .NET application (C#)
│   ├── src/          # Source files for .NET API
│   └── tests/        # Unit and integration tests
├── docs/              # Project documentation
└── README.md          # This file
```

## Technology Stack

### Frontend
- **Framework**: Angular (latest)
- **Language**: TypeScript
- **Package Manager**: npm/yarn

### Backend
- **Framework**: .NET (latest)
- **Language**: C#
- **Runtime**: .NET Core

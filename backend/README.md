# VoxBox Backend

.NET API application for VoxBox.

## Structure

```
backend/
├── src/              # Application source code
│   ├── VoxBox.Api/  # Main API project
│   ├── VoxBox.Core/ # Core business logic and domain models
│   ├── VoxBox.Infrastructure/ # Data access and external services
│   └── VoxBox.Shared/ # Shared utilities and common code
└── tests/           # Test projects
    ├── VoxBox.Api.Tests/
    ├── VoxBox.Core.Tests/
    └── VoxBox.Infrastructure.Tests/
```

## Technology Stack

- **Framework**: .NET (latest)
- **Language**: C#
- **Architecture**: Clean Architecture / Layered Architecture
